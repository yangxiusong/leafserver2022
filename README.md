此项目为基于：https://github.com/name5566/leafserver.git 的进行二次开发学习，采用go mod 方式进行构建，增加了基于tcp协议的几个小测试实例，提供了独立客户端测试代码并解决网络粘包问题


运行服务器：

cd server

go run main.go


运行客户端：

cd client

go run main.go


Leaf server
===========
A game server based on [Leaf framework](https://github.com/name5566/leaf).
